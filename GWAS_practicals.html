Description of the columns:

  1. ID: id code of sample
  2. sex: sex, 0 is female, 1 is male
  3. age: age, in years
  4. dm2: diabetes mellitus type 2, 0 is unaffected, 1 is affected
  5. height
  6. wieght
  7. diet
  8. bmi: BMI, body mass index, in \(cm/kg^2\)

It is important to check whether your trait of interest is correlated with covariates. To eheck this you can estimate the Peasron correlation coefficient and/or perform linear regression analysis. First, lets examine the correlation structure for phenotype data

cor_matrix <- cor(pheno[,c('dm2','sex','age','height','weight','bmi','diet')], use='complete.obs')
